This is app that give you information about required subnetwork. Just use request with subnetwork payload to get info about it. Example: 
  curl -d '192.168.1.1/30' http://$SERVER_ADDRESS/calc

    Finally, an ip address is available: 192.168.1.2
    Available ip address range: 192.168.1.1 ~ 192.168.1.2
    Address mask: 255.255.255.252
